### How To Use :-

####    Step 1 :- 
 Download the zip file or clone the repo
####    Step 2 :- 
In terminal type :
```
yarn
```

#### Step 3 :-
```
yarn start
npm run json-server
```

### Create a repo in github and add your project to the repo as following :- :-
