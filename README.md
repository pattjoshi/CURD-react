### How To Use :-

####    Step 1 :- 
 Download the zip file or clone the repo
####    Step 2 :- 
In terminal type :
```
yarn
```

#### Step 3 :-
```
yarn start
npm run json-server
```

git remote add origin https://github.com/pattjoshi/CURD-react.git
git push -u origin master
